# WIKI for Nobel Prize and Pulitzer Prize 搜索引擎

----
> scrapy+elasticsearch+flask+vue

## step 1 安装[中文分词插件ik](https://github.com/medcl/elasticsearch-analysis-ik)

- `elasticsearch-plugin install <>`

## 启动elasticsearch

- `bin/elasticsearch`

## 设置python环境

- `pip install -r requirements.txt`

## 启动backend

- `python app.py`

## 启动frontend

- `npm install`
- `npm run serve`

## 访问

- `ip:8080`
